# BOQ & Cost Control System

ระบบจัดการ BOQ และควบคุมต้นทุน สำหรับบริษัทรับสร้างบ้าน

------------------------------------------------------------------------

## 🎯 เป้าหมายของระบบ

-   รู้ต้นทุนก่อนขาย
-   คุมต้นทุนระหว่างก่อสร้าง
-   รู้กำไรจริงเมื่อจบโครงการ
-   ป้องกันงบประมาณบานปลาย

------------------------------------------------------------------------

# 1️⃣ BOQ คืออะไร

BOQ (Bill of Quantities) คือรายการวัสดุ + ปริมาณ + ราคาต่อหน่วย +
ค่าแรง\
รวมกันเป็นต้นทุนทั้งหมดของโครงการก่อสร้าง

ตัวอย่าง:

-   ปูน 500 ถุง × 120 บาท\
-   เหล็ก 2,000 กก. × 25 บาท\
-   ค่าแรง 300,000 บาท

------------------------------------------------------------------------

# 2️⃣ โครงสร้างระบบ

Project\
└── BOQ\
    └── Work Category\
        └── BOQ Item

------------------------------------------------------------------------

# 3️⃣ ข้อมูลใน BOQ Item

แต่ละรายการควรมี:

-   ชื่อรายการ
-   หมวดงาน
-   ปริมาณ
-   หน่วย
-   ราคาต่อหน่วย (ประมาณการ)
-   ต้นทุนรวม (คำนวณอัตโนมัติ)
-   ราคาจริง
-   ส่วนต่าง (Variance)

------------------------------------------------------------------------

# 4️⃣ โหมดการทำงาน

## 🟢 Estimated Cost

ใช้ตอนเสนอราคา

## 🔴 Actual Cost

อัปเดตต้นทุนจริงระหว่างก่อสร้าง

ระบบควรแสดง:

งบประมาณรวม\
ต้นทุนใช้จริง\
ส่วนต่างงบประมาณ

------------------------------------------------------------------------

# 5️⃣ สูตรคำนวณสำคัญ

ต้นทุนรวมหมวด = sum(BOQ Items)

ต้นทุนรวมโครงการ = sum(ทุกหมวด)

กำไร = ราคาขาย − ต้นทุนจริง

\% กำไร = (กำไร / ราคาขาย) × 100

------------------------------------------------------------------------

# 6️⃣ Dashboard ที่ควรมี

-   ต้นทุนรวมโครงการ
-   ต้นทุนใช้จริง
-   กำไรปัจจุบัน
-   หมวดงานที่เกินงบ
-   กราฟ Estimated vs Actual

------------------------------------------------------------------------

# 7️⃣ ฟีเจอร์ระดับมืออาชีพ

## ✅ Version Control

รองรับ BOQ หลายเวอร์ชัน (V1, V2, V3)

## ✅ Approval Flow

มีขั้นตอนอนุมัติก่อนใช้งานจริง

## ✅ Cost Alert

แจ้งเตือนเมื่อเกินงบที่กำหนด
